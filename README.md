# wjtools

Web 项目常用基础工具集，由日常开发项目提炼而来，支持按需引入，轻量灵活；扩展性强，可以方便地基于现有工具实现二次开发。

### 开发状态

开发中...

### 安装

```shell
npm install wjtools --save
```

### 使用

```js
import Wjtools from 'wjtools'
```

#### 按需使用

```js
import {
  Cookie,
  Storage,
  Ajax,
} from 'wjtools'
```
